# AI Resume Analysis ATS Scoring Recruitment Platform

Starter scaffold for a Streamlit-based recruitment platform that analyzes resumes, scores ATS compatibility, matches candidates to job descriptions, and generates reports.

## Structure

- `app.py` - Main Streamlit application
- `pages/` - Multi-page Streamlit screens
- `utils/` - Parsing, scoring, ranking, and reporting helpers
- `config/` - App and database configuration
- `database/` - Schema and local SQLite database
- `reports/` - Generated PDF, CSV, and Excel outputs
- `data/` - Uploaded and processed inputs
- `docs/` - Project documentation and screenshots

## Run

```bash
streamlit run app.py
```
