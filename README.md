## 📖 Project Overview

**AI Resume Analysis and Recruitment Platform** is an AI-powered recruitment application designed to simplify and automate the hiring process.

The platform helps recruiters analyze resumes, calculate ATS scores, compare resumes with job descriptions, rank candidates, generate interview questions, and create professional resumes using AI.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/AI-Resume-Analysis-and-Recruitment-Platform.git

cd AI-Resume-Analysis-and-Recruitment-Platform
```

---

## Create Virtual Environment

```bash
python -m venv venv
```

Activate

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

---

## Install Requirements

```bash
pip install -r requirements.txt
```

---

## Run Streamlit

```bash
streamlit run app.py
```

---

# 🐳 Docker

Build

```bash
docker compose build
```

Run

```bash
docker compose up
```

Open

```
http://localhost:8501
```
